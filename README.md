#AOP
    
    切面：横切关注点被模块化的特殊对象
    通知：切面必须完成的工作  切面里面的验证，方法等
    目标：被通知的对象 AtithmeticCalculatorImpl 这个类的对象
    代理：proxy
    连接点:程序执行的某个特定的位置，方法前 方法后等
    切点：每个类有多个连接点，aop通过切点定位到特定的连接点
